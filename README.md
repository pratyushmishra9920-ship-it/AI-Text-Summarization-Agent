A professional multi-mode text summarization system powered by **Gemini 2.5 Flash**, capable of generating short, medium, long, bullet-style, simple, and professional summaries with **keyword extraction, sentiment detection, and a 1-line takeaway**.  
Designed & implemented as my first AI project after completing **Google 5-Day AI Intensive ‚Äî Capstone Project**.

---

## üöÄ Features
- Multiple summarization modes (Short, Medium, Long, Bullets, Simple, Professional)
- Automated keyword extraction
- Sentiment analysis (Positive / Neutral / Negative)
- One-line final takeaway
- Clean CSV Export
- Professional PDF Report Generator (with watermark, headings, and layout)
- Built in Kaggle using Python & ReportLab
